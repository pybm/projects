## Embedded Lab

A collection of embedded projects developed with MicroPython, combining programming with hands-on hardware work.

I experimented with Raspberry Pi Pico and ESP32, 

using components like TFT displays, 

LEDs, relays, and UART communication to build practical and interactive systems.

Main projects:

- Hamsterul cu WiFi  

- RF Intercom Control  

- Rock Paper Scissors (Potentiometer-Based)
